# React Native + Firebase = Instagram Clone ❤️

This project is a fully functional Instagram Clone built using **React Native** and **Firebase**. The app includes features like user authentication, photo/video uploads, real-time feeds, likes, comments, and more. It is designed to mimic the core functionality of Instagram with seamless performance and a clean user interface.

## Key Features:

- **User Authentication**: Sign up and log in using Firebase Authentication.
- **Image & Video Uploads**: Upload posts with photos or videos.
- **Real-Time Feed**: See the latest posts from users in real-time using Firebase Firestore.
- **Likes & Comments**: Interact with posts by liking and commenting.
- **Profile Management**: Customize user profiles, including bio and profile picture.
- **Push Notifications**: Receive notifications for likes, comments, and follows.

This project demonstrates the power of combining **React Native** for cross-platform mobile development with **Firebase** for backend services.

## Technologies Used

- **React Native**: Cross-platform mobile framework
- **Firebase**: Backend services (Authentication, Firestore, Storage, etc.)
- **Expo or React Native CLI**: Depending on the development environment
